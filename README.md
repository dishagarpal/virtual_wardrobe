# Virtual Wardrobe

A stylish web application that helps users organize and manage their clothing collection digitally.

## Features

* User Registration & Login
* Add clothing items with images
* Store brand, color, category, season, occasion, and price
* Search clothes by name, brand, or color
* Filter clothes by category
* Mark items as favorites ❤️
* Delete clothing items
* Persistent data storage using Local Storage
* Responsive and modern Myntra-inspired UI

## Technologies Used

* HTML5
* CSS
* JavaScript
* Local Storage API
* FileReader API

## Concepts Implemented

* DOM Manipulation
* Event Handling
* Authentication System
* Local Storage
* JSON Parsing & Stringifying
* Dynamic Rendering
* Array Operations (filter, map, forEach)
* Responsive Web Design
